Project Overview

This project aims to enhance the 3D Slicer open-source medical imaging platform by enabling African scientists and developers to create AI-powered extensions in local languages. The initiative focuses on accessibility, localization, and the integration of customized artificial intelligence models in healthcare workflows.

Objectives

  • Enable multilingual and right-to-left interfaces in 3D Slicer extensions
  • Leverage Large Language Models for interface translation
  • Empower African researchers and developers through training
  • Support AI-based medical imaging solutions for local health challenges

Use Case Example

An early application of this project targets AI-assisted diagnosis of mycetoma, a neglected tropical disease. The solution aims to be usable in English and Arabic-speaking contexts, supporting broader accessibility.

Funding & Collaboration

This project is funded by the Wellcome Trust and led by an international collaboration involving Brigham & Women’s Hospital / Harvard Medical School, alongside partners across Africa, Europe, and North America.
